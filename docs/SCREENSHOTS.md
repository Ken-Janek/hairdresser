# Application Screenshots

## 1. Booking Interface
Main booking page where users can:
- Select a date and stylist
- Choose a service
- View available time slots
- Enter customer details (name, email, phone)
- Confirm their booking

**File:** `booking.png`

## 2. Booking Confirmation
Confirmation page displayed after successful booking:
- Shows stylist name
- Shows service and cost
- Displays booking date and time
- Confirms customer email

**File:** `confirmation.png`

## 3. Admin Bookings View
Admin interface for managing bookings:
- Filter bookings by date
- View all bookings in table format
- Shows time, client name, stylist, service, and contact info
- Cancel button for each booking
- Protected with HTTP Basic Authentication

**File:** `admin.png`

## How to Add Screenshots

1. Take screenshots of each interface
2. Save them as PNG files in the `docs/` folder
3. Update paths in README.md if needed
